# GestureX-Power-Battle-System

An interactive AI-based hand gesture battle system built using OpenCV and MediaPipe.

Features
- 🔥 Fire attack using hand gestures
- ⚡ Lightning attack with charging effect
- ✨ Real-time hand tracking
- 💥 Explosion & camera shake effects
- 🎮 Two-player interaction system
  
- Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

How it works
- Left side → FIRE 🔥
- Right side → LIGHTNING ⚡
- Hold gesture to charge → release attack
